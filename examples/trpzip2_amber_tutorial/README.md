## trpzip2 AMBER tutorial example

This folder contains a small example dataset for ShineMD based on the official AMBER CPPTRAJ Tutorial C1:

- `trpzip2.ff10.mbondi.parm7`
- `trpzip2.gb.nc`
- `trpzip2.1LE1.1.rst7`

Primary source:

- AMBER CPPTRAJ Tutorial C1, "RMSD Analysis in CPPTRAJ"
  https://ambermd.org/tutorials/analysis/tutorial1/

Why it is included here:

- quick demo dataset for ShineMD
- validation target for `.parm7` topology support
- reference case for comparing ShineMD RMSD behavior against the published cpptraj tutorial workflow

Provenance:

- `trpzip2.gb.nc`, `trpzip2.ff10.mbondi.parm7`, and `trpzip2.1LE1.1.rst7` were copied from the AMBER tutorial materials linked above

Notes:

- These files are third-party tutorial materials and are included here with source attribution for reproducibility and validation purposes.
- The repository's MIT license applies to ShineMD source code; the example dataset retains its original provenance.
